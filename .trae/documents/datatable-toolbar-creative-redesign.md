# 数据表工具栏创意设计重构计划

## 一、当前实现分析

当前 `DataTableToolbar.tsx`（381行）是一个传统风格的工具栏：
- 左侧：两个独立输入框（Project ID / Test Run ID）+ 搜索按钮
- 右侧：密度选择器、视图模式下拉、列可见性按钮、历史记录按钮、更多操作菜单
- 密封模式下：静态面包屑显示
- 样式：标准按钮+边框，无动画特色，交互平淡

**问题**：布局传统无趣，"更多"菜单隐藏了重要操作，缺少微交互和视觉个性，整体体验缺乏活力。

---

## 二、新设计概念：「Mission Control」指挥中心风格

将工具栏从传统条状控件升级为 **沉浸式指挥中心** 风格，核心设计语言：**流动、发光、响应式**。

### 2.1 统一指令输入栏（Command Bar）

**替代**：两个独立输入框 → 一个统一的智能指令栏

- 单一宽幅输入栏，带有呼吸光标动画
- 占位符使用打字机效果（typewriter）循环展示：`输入 Project ID 和 Test Run ID 开始探索...` → `试试: project-123 / run-456` → `按 Enter 发起查询`
- 智能解析：用户输入 `project-123 / run-456` 自动拆分填充两个字段
- 输入栏聚焦时展开发光边框（accent color glow），失焦时收缩变淡
- 搜索按钮融合进输入栏右侧，变成一个圆形发射按钮（▶），点击时有波纹扩散效果
- 加载中时，输入栏底部显示一条流光进度条（shimmer wave）

### 2.2 轨道式控件环（Orbital Controls）

**替代**：右侧散落的各种按钮 → 紧凑的轨道式控件组

- 控件排列为一条紧凑的图标轨道，每个图标按钮：
  - 默认状态：小圆点图标 + 微弱边框
  - 悬停状态：图标放大 1.15x + 发光光晕 + tooltip 浮现
  - 激活状态：accent 色填充 + 脉冲动画
- 图标之间用极细的虚线连接（轨道感），而非实线分隔符
- 控件顺序：密度 → 视图模式 → 列 → 历史 → 导出 → 刷新

### 2.3 液态密度切换器（Liquid Density Toggle）

**替代**：三按钮密度组 → 单按钮循环切换 + 液态变形动画

- 一个按钮，点击循环切换 comfortable → standard → compact
- 图标在切换时有平滑的 morph 变形动画（行数从3行→2行→1行的过渡）
- 当前密度以小标签形式显示在图标下方（如 "STD"），切换时标签有翻转动画
- 支持键盘左右方向键快速切换

### 2.4 分段式视图模式（Segmented View Mode）

**替代**：下拉菜单 → 滑动分段控件

- 两个选项 Simple / Detailed 排列在胶囊形容器中
- 选中项有一个滑动高亮块（sliding highlight），切换时高亮块平滑滑动到目标位置
- 高亮块带有微妙的弹性动画（spring easing）

### 2.5 上下文感知操作芯片（Context Chips）

**替代**："更多"下拉菜单 → 动态浮入的操作芯片

- 不再使用 More 菜单，操作以芯片（chip）形式根据上下文动态出现：
  - **导出芯片**：数据加载后滑入显示，点击展开为 CSV/JSON/Excel 三个子芯片
  - **刷新芯片**：始终可见，加载中时变为旋转动画
  - **预设芯片**：点击展开 ViewPresetManager
- 芯片出现时有 `slideIn + fadeIn` 动画，消失时 `slideOut + fadeOut`
- 每个芯片带有微弱的 accent 色左边框

### 2.6 密封模式：动画面包屑轨迹

**替代**：静态面包屑 → 绘制式动画轨迹

- Project ID 和 Test Run ID 之间有一条动画连接线（从左到右绘制）
- 每个节点有脉冲点效果
- 整体呈现「数据流」的视觉效果

### 2.7 全局微交互

- **磁吸悬停**：按钮在鼠标靠近时微微向鼠标方向偏移（2px）
- **波纹点击**：所有可点击元素点击时产生从点击位置扩散的波纹
- **加载呼吸**：工具栏在加载状态下有微弱的呼吸灯效果（opacity 在 0.85-1.0 间脉动）
- **成功闪烁**：查询成功后，工具栏底部边框短暂闪烁 accent 色
- **键盘快捷键提示**：悬停按钮时，tooltip 中显示快捷键（如 `⌘K` 搜索）

## 五、设计原则

1. **流动感**：所有状态切换都有平滑过渡，拒绝生硬跳变
2. **发光感**：accent 色作为光源，聚焦/激活时产生光晕
3. **响应感**：每个操作都有即时视觉反馈
4. **克制感**：动画时长 140-240ms，不拖沓不炫技
5. **一致性**：所有交互遵循统一的动画曲线和时序
