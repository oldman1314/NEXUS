# Workflow UI 重构 Checklist

- [x] 浏览器原生 prompt() 已被替换为内联保存对话框
- [x] 保存对话框包含名称和描述输入框，风格与应用一致
- [x] 已有工作流保存时直接更新，另存为时弹出对话框
- [x] 节点面板支持拖拽添加节点到画布指定位置
- [x] 点击添加节点时基于视口中心定位，非随机位置
- [x] 连续添加节点自动偏移避免重叠
- [x] API 节点配置面板包含 Headers 键值对编辑器
- [x] API 节点配置面板包含 Params 键值对编辑器
- [x] Headers/Params 数据正确同步到节点 data 并在执行时生效
- [x] Transform 节点使用带行号的代码编辑器（复用项目 useCodeEditor 模式，真语法高亮需引入 CodeMirror 等库）
- [x] 侧边栏点击工作流自动切换到 Workflow 视图并加载
- [x] 编辑器中加载的工作流在侧边栏高亮显示
- [x] 画布为空时显示引导信息和快速操作入口
- [x] 预置模板可正常创建完整工作流
- [x] 工作流执行时活跃边线显示流动动画
- [x] 日志面板高度可通过拖拽调整
- [x] Delete/Backspace 键可删除选中节点
- [x] Ctrl+S 可保存工作流
- [x] Escape 可取消选中或关闭面板
- [x] 条件节点 True/False Handle 有明确的颜色区分和标签
- [x] 工作流描述可在保存对话框中编辑
- [x] 所有新增组件的动画和交互与现有设计语言一致
